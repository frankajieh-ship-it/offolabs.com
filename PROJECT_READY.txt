                      COMMON QUESTIONS

Q: Is the site already running?
A: No, you need to run "npm install" then "npm run dev" first.

Q: What should I edit first?
A: Start with app/components/sections/ - add new section components there.

Q: How do I add pages?
A: Create app/your-page/page.tsx and wrap content in MainLayout.

Q: Can I use my own CSS?
A: Yes, but prefer Tailwind classes. Use globals.css for global styles.

Q: How do I deploy?
A: See SETUP_INSTRUCTIONS.md â†’ Deployment section.

Q: What if I get errors?
